### 🚀 Flagship Project Spotlight

#### [🛡️ SurveilX: AI-Powered Smart Surveillance Ecosystem](https://github.com/abdulah-naeem/SurveilX)
> A scalable, production-ready video surveillance dashboard integrating real-time threat analysis and semantic footage search.
- **Real-Time Violence Detection:** Built a highly optimized **CNN-LSTM Fusion** pipeline to monitor live multi-threaded camera feeds (15 FPS) and trigger real-time physical security alerts.
- **Natural Language Footage Retrieval:** Engineered a forensic search engine powered by **OpenAI CLIP** embeddings and **ChromaDB** vector storage to instantly locate events using free-form textual queries.
- **Adaptive Processing Architecture:** Incorporated a custom streaming backend using **FastAPI**, relational persistence via **PostgreSQL**, and automated global CDN sync with **Cloudinary**.
